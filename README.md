# DHR Web Studio

Portfólio oficial da **DHR Web Studio**, criado para apresentar serviços, projetos, planos e canais de contato da marca.

## Sobre o projeto

O site reúne:

- apresentação da DHR Web Studio;
- serviços de landing pages, sites institucionais, portfólios e reformulação;
- planos comerciais de landing pages e sites;
- portfólio interativo com projetos reais;
- formulário integrado ao WhatsApp;
- aviso e definições de cookies;
- layout responsivo para desktop, tablet e celular.

## Tecnologias

- HTML5
- CSS3
- JavaScript

O projeto não depende de framework ou processo de build.

## Executar localmente

Clone o repositório e abra a pasta em um servidor local. Com Python instalado:

```bash
python -m http.server 4173
```

Depois, acesse `http://127.0.0.1:4173` no navegador.

> Para que a visualização interna dos projetos funcione corretamente, use um servidor local em vez de abrir o arquivo `index.html` diretamente.

## Estrutura principal

```text
assets/     Imagens, logos e capas dos projetos
css/        Estilos e responsividade
js/         Configurações, projetos e interações
projects/   Versões locais dos projetos apresentados no portfólio
index.html  Página principal
```
