# Ilumine Semijoias

Landing page moderna para a Ilumine Semijoias, com apresentacao da marca, colecoes, cuidados com as pecas, avaliacoes de clientes e uma pequena loja integrada ao WhatsApp.

## Recursos

- Layout responsivo para desktop e celular.
- Vitrine de produtos com valores.
- Carrinho lateral com quantidade, total e remocao de itens.
- Finalizacao pelo WhatsApp com mensagem automatica contendo os produtos escolhidos.
- Secoes de colecoes, sobre a marca, cuidados, momentos especiais, Instagram e contato.
- Favicon personalizado com o diamante da marca.

## Como Rodar Localmente

Abra a pasta do projeto e execute:

```bash
node dev-server.js
```

Depois acesse:

```text
http://127.0.0.1:4173/
```

Como alternativa, por ser um site estatico, tambem e possivel abrir o arquivo `index.html` diretamente no navegador.
